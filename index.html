Atenção

Branding/Logo

Apesar de opcional, esta configuração é interessante para o cliente pois aplica a marca dele no aplicativo. Esta configuração está disponível no menu da engrenagem, na tela de login. Seu ícone é representado por uma imagem.

Atenção

Esta configuraçao necessita da senha administrativa para ser alterada. Se voce nao e um administrador do aplicativo, contate um representante H&S.

Ao inserir a senha, você poderá escolher uma imagem PNG do logo do cliente. Esta imagem substituirá a imagem padrão do aplicativo

Limpeza de Dados

Caso seja necessário, o usuário pode limpar os dados offline do aplicativo. Entende-se por dados offline tudo aquilo que foi copiado do servidor para o dispositivo. Esta configuração está disponível no menu da engrenagem, na tela de login. Seu ícone é representado por uma lata com um X.

Atenção

Ao executar este procedimento NÃO SERÁ POSSÍVEL realizar o login fora da fábrica. Será necessário recuperar os dados de login, veículos, linhas e tanques na fábrica.

Para evitar acidentes com os dados, uma confirmação por texto é necessária.

Login Atualizado em: 2022-05-04

Para realizar coletas é necessário fazer o login toda vez que o app for aberto. o ultimo usuário sempre virá preenchido automaticamente, necessitando somente da senha para entrar. Se estiver fora da fábrica, não é possivel trocar o usuario logado.

Os login subsequentes podem ser realizados desconectado da rede ou da internet. Caso esteja conectado á fabrica, os login subsequentes também atualizarão os dados (inclusive sua senha, caso tenha solicitado alteração ao administrador).

Credenciais

Para acessar o App, é necessário liberação por parte do administrador. Após liberação, voce pode acessar o app com seu CPF, CNPJ ou Email cadastrados no sistema SCL.

Atualização de Dados

O primeiro login deve ser feito dentro da fábrica, com o dispositivo conectado na mesma rede do servidor. Este primeiro login é importante pós além de validar as credenciaisdo motorista/carreteiro, também vai atualizar os dados do dispositivo (Veículos, Linhas e Tanques).

Os logins subsequentes podem ser realizados desconectado da rede ou da internet. Caso esteja conectado á fabrica, os login subsequentes também atualizarão os dados (inclusive sua senha, caso tenha solicitado alteração ao administrador).

Recuperação de Senha

Caso tenha esquecido a senha, solicite ao administrador uma nova senha. ela será sincronizada com o dispositivo quando se conectar novamente com a fabrica

Atenção

Não existe processo de recuperação de senha enquanto o motorista não está conectado a fábrica. Esquecer a senha pode impedir a realização de uma coleta.

Coletas Abertas

O motorista pode retomar uma coleta aberta. Caso opte por começar do zero, os dados de coleta atuais são limpos e uma nova coleta será iniciada.
